# 🌙 Sore Hening — Deploy ke Vercel

## Cara Deploy (5 Menit, Gratis)

### Metode 1: Drag & Drop (Termudah)
1. Buka https://vercel.com/new
2. Login dengan Google/GitHub
3. Klik "Browse" → pilih folder `sore-hening-vercel` ini
4. Klik Deploy → selesai! 🎉

### Metode 2: Via Terminal (1 Perintah)
```bash
npx vercel --yes
```
Jalankan di dalam folder `sore-hening-vercel` ini.

### Metode 3: Via GitHub
1. Upload folder ini ke GitHub repo baru
2. Connect repo ke Vercel
3. Auto-deploy setiap kali update

---
Website akan live di: https://sore-hening.vercel.app (atau nama custom)
